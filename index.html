Catching Fireflies

About the Game

This game is about catching 'Fireflies' which are the yellow dots and avoid the red bees. The aim is to get the highest score possible. I chose this game because it will be a good example to utilise the various skills I had learnt in this module for the semester. This is a simple game suited for kids, thus, the music used is also generally more cheerful and encouraging.


I used the code for one of our assignment as the base and work on it to develop a new game. Hence, the layout is similar to the game assignment. However, the aesthetics were modified to suit the game.

I used 2 different arrays of circles to depict the 2 different characters in the game which are the fireflies and bees recpectively. There are more circles in the array for fireflies so that the player will be able to 'catch' the fireflies and avoid the bees.

I used gravity to make the circles fall from top to bottom of the canvas and randomised the x position where the circles will start to fall so that they will fall at the same height but is still spreaded out. The rates at which they fall are also randomised so that the game will be more fun.

The score is recorded at the bottom of the page so that players can keep track of their scores. Each round will run for 30 seconds. This is done using a timer which will start only when the 'play' button is clicked.

To catch the fireflies, I used a event listener so that the 'net' will follow the mouse where it goes. Whenever the position of the mouse and the fireflies (which is the array of circle) is within the distance of 50px, the firefly will be considered caught and 10 points is given to the player. Using a similar concept, when the mouse position is within distance range of 50px with the 'bees', 5 points will be deducted through minusing the counter
